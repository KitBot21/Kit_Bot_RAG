# Ground Truth 생성 가이드

## 🎯 기본 원칙

**"사용자에게 가장 유용한 정보가 정답"**

---

## 📏 선택 규칙

### 1️⃣ 최신성 우선

**같은 내용이라면 최신 정보 선택**

```
쿼리: "통학버스 출발시간"

검색 결과:
[1] 2024-2학기 대구통학버스 운행 노선표  ← 선택! ✅
[2] 2024-1학기 대구통학버스 운행 노선표  
[3] 2023-2학기 대구통학버스 운행 노선표  

→ 1번 선택 (가장 최신)
```

**우선순위:**
- 2024-2학기 > 2024-1학기 > 2023-2학기 > ...
- 2학기 > 1학기 (같은 연도)

---

### 2️⃣ 내용 완전성 우선

**정보가 더 완전하고 구체적인 것 선택**

```
쿼리: "기숙사 식당 운영시간"

검색 결과:
[1] 생활관 안내 (짧음)
    "식당: 오전 8시~오후 7시"
    
[2] 2024학년도 생활관 운영 안내 (상세함)  ← 선택! ✅
    "조식: 08:00~09:00
     중식: 12:00~13:30
     석식: 17:30~19:00
     토/일/공휴일: 중식/석식만 운영"

→ 2번 선택 (더 구체적)
```

---

### 3️⃣ 직접 답변 우선

**쿼리에 직접 답하는 내용 선택**

```
쿼리: "수강신청 최대 몇 학점?"

검색 결과:
[1] 학사 규정
    "제17조 수강신청 학점은 학기당 최대 21학점..."  ← 선택! ✅
    
[2] 수강신청 안내
    "수강신청 방법: 포털 로그인 → 수강신청 메뉴..."
    
[3] 학사일정
    "2024-2학기 수강신청: 8월 20일~22일"

→ 1번 선택 (직접 답변)
```

---

### 4️⃣ 공식 문서 우선

**비공식 정보보다 공식 문서 선택**

```
쿼리: "전과 가능한가요?"

검색 결과:
[1] 2024년 전과 전출입 선발 방법.xlsx  ← 선택! ✅
    (공식 문서)
    
[2] 게시판 질문
    "전과 문의드립니다..."
    
[3] 학과 소개 페이지
    "...전과 제도가 있습니다..."

→ 1번 선택 (공식)
```

---

## ⚠️ 애매한 경우

### Case 1: 여러 정답이 있을 때

**모두 유용하면 가장 포괄적인 것 선택**

```
쿼리: "장학금 종류"

[1] 국가장학금 안내
[2] 교내장학금 안내
[3] 2024 장학금 전체 안내  ← 선택! ✅

→ 3번 (전체 포함)
```

---

### Case 2: 정답이 없을 때

**관련 정보도 없으면 `n` (No Answer)**

```
쿼리: "동아리 회비는 얼마인가요?"

검색 결과:
[1] 동아리 소개
[2] 동아리 가입 방법
[3] 학생회 회비

→ n 입력 (정답 없음)
```

---

### Case 3: 확실하지 않을 때

**나중에 다시 확인하려면 `s` (Skip)**

```
쿼리: "복수전공이랑 부전공 차이?"

검색 결과 모두 애매함...

→ s 입력 (스킵)
```

---

## 🔄 우선순위 요약

1. **최신성** (2024 > 2023)
2. **완전성** (상세한 정보 > 짧은 정보)
3. **직접성** (직접 답변 > 간접 언급)
4. **공식성** (공식 문서 > 게시판)
5. **포괄성** (전체 안내 > 부분 안내)

---

## 💡 팁

### 빠르게 판단하는 법

1. **제목 먼저 보기**
   - 연도 확인 (최신?)
   - 문서 종류 (공식?)

2. **내용 스캔**
   - 쿼리 키워드 포함?
   - 구체적 정보 있음?

3. **점수 참고 (절대적 기준 아님)**
   - 0.7 이상: 관련성 높음
   - 0.5~0.7: 부분 관련
   - 0.5 미만: 관련 낮음

---

## ⏱️ 시간 관리

- **한 쿼리당 30초~1분** 목표
- 10개 완료 → 자동 저장 ✅
- 피곤하면 → `q` 종료 후 나중에 재개

- [ ] 공식 문서인가?
